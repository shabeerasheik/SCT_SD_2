# рџ“Њ Project Overview

The **Random Number Guessing Game** is a desktop-based Java application developed using **Java Swing** as part of the **SkillCraft Technology Software Development Internship (Task 02)**.

The application generates a random number between **1 and 100**, allowing users to guess the correct number through an interactive graphical interface. It provides intelligent feedback such as **Too High** and **Too Low**, validates user input, tracks the number of attempts, and allows users to start a new game after successfully guessing the number.

This project demonstrates **Object-Oriented Programming**, **Event Handling**, **GUI Development**, **Random Number Generation**, and **Exception Handling** in Java.
